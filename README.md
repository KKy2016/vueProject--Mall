# vueProject
在学习vue后，自己模仿电商网站，使用vue实现了基本的功能与交互效果，希望大家可以一起学习与分享
如果大家发现有错误或者建议,欢迎大家砸鸡蛋

 # 启动项目

 # npm install  下载依赖

 # npm run mock  

 # npm run dev

 //使用vuex实现多组件之间数据通信，
   
   vue-router实现路由跳转，
   
   vue-resource进行数据请求，
   
   vue-lazyload实现图片懒加载，
   
   json-server插件实现mock数据
